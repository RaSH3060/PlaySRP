Гайд по использованию PlaySRP для Steam Remote Play

1. Установка
Поместите файл PlaySRP.exe в папку с игрой внутри Steam-директории (пример: D:\Games\Steam\steamapps\common\Игра\).

Переименуйте PlaySRP.exe в название исполняемого файла игры (например, Игра.exe).

Создайте в этой же папке текстовый файл play.ini.

2. Настройка play.ini
Формат записи:

play|Полный_путь_к_файлу|Параметры|Кастомное_имя 

Примеры:

play|C:\Program Files\MyGame\game.exe|-windowed|Моя Игра
play|D:\Игры\Best game\bin\game.exe||CoolGame


Drag-and-drop: Перетащите любой .exe-файл на PlaySRP.exe, чтобы автоматически добавить его путь в play.ini.

3. Добавление параметров запуска
Нажмите ПКМ на строку с выбранной игрой в PlaySRP → выберите Edit Parameters.
В появившемся окне введите нужный параметр (например, -windowed).

4. Запуск программ

Запустите переименованный PlaySRP.exe через Steam.
Выберите программу из списка → нажмите Play.


Выберите программу из списка → нажмите Play.

5. Особенности работы
Steam Overlay работает всегда (активация: Shift+Tab).

Remote Play:

Демонстрация экрана и управление для других игроков.

Совместимость зависит от приложения:
Заменяемое приложение должно поддерживать Steam Remote Play

Примеры:

Cube Racer — нет, Поддерживает но не работает, есть исключения!
Mini Thief — да,
Half-Life 2 — да.

6. Решение проблем
Ошибка запуска:

Проверьте корректность пути в play.ini.

Убедитесь, что файл существует.
